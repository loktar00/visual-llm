Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build an endlessly looping Rube Goldberg machine viewed side-on: a ball rolls down ramps, tips dominos, rides a pulley basket, triggers a spring hammer, sails through a funnel, and finally returns to the start via an elevator wheel, closing the loop perfectly. All steps need real collision and constraint physics, not canned animation. The camera pans to follow the action. Small dust puffs and squash-stretch accents sell impacts.
