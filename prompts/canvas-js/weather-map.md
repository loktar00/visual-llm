Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build an animated fictional weather map: a coastline continent with a wind vector field visualized by thousands of flowing particles, pressure systems that drift and spin the wind around highs and lows, rain bands rendered as sweeping translucent texture, and temperature as a soft underlying color ramp. Fronts crawl with standard triangle-and-bump notation. Systems evolve continuously so the map never repeats. No controls.
