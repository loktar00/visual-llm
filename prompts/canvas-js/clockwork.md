Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build a screensaver of an ever-running clockwork machine: interlocking gears of many sizes with correct meshing ratios, a swinging escapement, chains over sprockets, and a piston or two, all driving a small automaton bird that pops out hourly (every 45 seconds here) to flap and chirp silently. Occasionally a gear pops loose, bounces off others with physics, and a repair arm re-seats it. Brass-and-iron palette.
