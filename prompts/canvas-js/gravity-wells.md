Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build a gravity-painting screensaver: streams of thousands of tiny particles continuously emit from the screen edges and fall through a field of 3 to 5 invisible gravity wells that slowly wander, curving the streams into orbital ribbons, slingshots, and captures. Particles leave additive trails that build glowing composite paintings; fade the canvas slowly so the artwork keeps evolving. Wells occasionally merge or split. No interaction.
