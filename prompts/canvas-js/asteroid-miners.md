Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build a top-down autonomous asteroid-mining sim: 5 hauler ships fly between a rotating station and a drifting asteroid belt, lock on with mining beams, chip glowing ore chunks free, tractor them aboard, and dock to unload. Asteroids split realistically and drift with momentum; depleted zones respawn elsewhere. Pirates occasionally raid and are driven off by two escort drones with laser fire. Camera pans the belt slowly.
