Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build a fireworks night over a city skyline reflected in a river. Launch shells on a randomized schedule with whistling trails, bursting into varied patterns: peonies, willows, rings, and double breaks with distinct palettes. Particles need gravity, drag, and fade-to-ember. The river reflection should shimmer. Fire a rapid finale barrage every few minutes. Endless, no controls.
