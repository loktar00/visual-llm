Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build a harbor scene at dusk: moored fishing boats bob on gentle swells with correct buoyancy tilt, gulls wheel and land on posts, a lighthouse sweeps its beam across the water with volumetric glow, and a ferry crosses the bay on a schedule, leaving a wake. Water needs layered waves, specular glints along the light path, and reflections that wobble. Sky slowly deepens through sunset colors on a loop.
