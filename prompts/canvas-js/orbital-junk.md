Create a single self-contained HTML file using only HTML, CSS, and JavaScript. No build tools or external assets. Use a full-screen 2D canvas.

Build an orbital debris screensaver: Earth in one corner, hundreds of satellites and junk fragments on distinct orbital rings with lower orbits moving faster. Occasionally two objects collide and burst into a spreading cloud of tracked fragments that inherit and perturb the parent orbit. A cleanup drone slowly chases and deletes strays so the sky never saturates. Faint orbit lines, tiny glinting sprites.
